# password-manager
## How to run locally
- Server
1. Run `cd server`
2. Run `npm install`
3. Run `node bin/www`
- Client
1. Run `cd client`
2. Run `npm install`
3. Run `npm run dev`
 
### Register / Login
- Login with register account
1. Input your email & password
2. Login
3. Verify to your email account
- Login with google account
1. Add your google account
2. Login
